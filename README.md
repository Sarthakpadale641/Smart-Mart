# 🛒 Smart-Mart

Smart-Mart is a simple e-commerce website project made using HTML, CSS, and JavaScript.

This project is created for learning purpose to understand how an online shopping website works.

---

## 📌 About The Project

Smart-Mart is a basic shopping website where users can:

- View products
- Login and Register
- Add products to cart
- Visit About page
- Use Forgot Password page

This project does not use backend or database. It is a front-end project only.

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript

---

## 📂 Project Files

- home.html → Main page
- login.html → Login page
- register.html → Register page
- cart.html → Shopping cart
- about.html → About page
- forgot.html → Forgot password page
- style.css → Styling file
- script.js → JavaScript functionality

---

## 🚀 How To Run This Project

1. Download or clone this repository:
   git clone https://github.com/Sarthakpadale641/Smart-Mart.git

2. Open the folder.

3. Double click on home.html to open in browser.

That’s it ✅
